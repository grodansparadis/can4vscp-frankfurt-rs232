====== Change Log for Frankfurt RS-232 ======

  * 2016-11-28 AKHE - Driver has changed name vscpl1drv_can4vscp.so is now vscpl1drv-can4vscp.so
  * 2015-10-08 AKHE - Implemented baudrate change command/configuration.
  * 2015-10-08 AKHE - Implemented hardware handshake.
  * 2015-10-08 AKHE - Added support for timestamp.
  * 2015-10-07 AKHE - Bus off/warning system did not work correctly. Fixed.
  * 2015-10-06 AKHE - Added **set defaults** command that set all persistent values to there default value.
  * 2015-10-06 AKHE - Changed default timeout to 20 ms.
  * 2015-10-05 AKHE - **rreg** and **wreg** now show nodeid info.
  * 2015-10-05 AKHE - Fixed problem with buffer overwrite.
  * 2015-10-04 AKHE - Fixed problem that caused messages with wrong crc to be written out.
  * 2015-09-30 AKHE - Fixed "dead adapter need to toggle power to work" problem.
